## Provenance Tracking

Every generated asset carries embedded provenance metadata:

- **MP3 files**: `TXXX:Lore_Ledger` ID3 tag contains a frozen `LicenseRecord`
  with generation timestamp, API versions, and usage terms
- **JPEG files**: EXIF `UserComment` contains the same license JSON
- **All metadata is static** — once stamped, it is never modified

## SynthID Watermarking

All audio generated by Google's Lyria API includes a native
[SynthID](https://deepmind.google/technologies/synthid/) watermark.
This invisible digital signature identifies the audio as AI-generated
and is preserved through format conversion and light editing.
